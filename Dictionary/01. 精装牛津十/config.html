OALDPE — 用户自定义配置
中文翻译相关
配置项1：中文翻译选项
（点击 “O10” 小图标，可显示/隐藏中文翻译）
0 - 全部隐藏
1 - 全部显示（默认）
2 - 仅隐藏例句中文
3 - 仅显示例句中文
4 - 仅隐藏释义中文
5 - 仅显示释义中文
配置项2：是否使用繁体中文翻译
0 - 简体（默认）
1 - 繁体（香港）
2 - 繁体（台湾）
3 - 繁体（台湾，带词组转换）
配置项3：是否启用英文点译功能
（单句显示/隐藏中文）
0 - 不启用（默认）
1 - 启用
配置项4：是否例句中文独占一行
0 - 不独占一行
1 - 独占一行（默认）
词性导航栏
配置项5：是否显示词性导航栏
0 - 不显示
1 - 显示（默认）
配置项6：是否选中词性导航栏 All
（当【是否显示词性导航栏】设置为不显示时，始终显示所有词性，此设置无效）
0 - 不选中（默认）
1 - 选中
单词、例句发音，图片显示
配置项7：是否显示音节划分
（点击单词可切换音节划分）
0 - 不显示（默认）
1 - 显示
配置项8：是否启用在线单词发音
（如果为 true，则可删除 oaldpe.1.mdd 文件）
0 - 不启用（默认）
1 - 启用
配置项9：是否启用在线图片
（如果为 true，则可删除 oaldpe.2.mdd 文件。注：在线图片无中文翻译）
0 - 不启用（默认）
1 - 启用
配置项10：离线图片翻译选项
（当【是否启用在线图片】启用时，图片翻译无效）
0 - 不使用翻译
1 - 简体中文翻译
2 - 港版繁体翻译
3 - 自动选择（默认）
配置项11：是否默认英音例句发音
（点击 “O10” 小图标中的词典铭牌进行切换）
0 - 美音（默认）
1 - 英音
配置项12：官方例句发音选项
（如果为 1 或 2，则可删除 oaldpe.3.mdd 文件）
0 - 启用官方离线例句发音
1 - 启用官方在线例句发音（默认）
2 - 不启用官方例句发音
配置项13：是否删除无在线发音的官方例句发音
（删除 oaldpe.3.mdd 文件时，该部分例句无法发音，可转而使用 TTS 发音）
0 - 不删除（默认）
1 - 删除
配置项14：无官方例句发音时，是否启用在线 TTS 发音
（需要高版本浏览器内核。发音图标带下划线）
0 - 不启用
1 - 启用（默认）
配置项15：TTS 英音发音配置
0 - 英音男1（默认）
1 - 英音男2
2 - 英音女1
3 - 英音女2
4 - 英音女3
配置项16：TTS 美音发音配置
0 - 美音男1
1 - 美音男2
2 - 美音男3
3 - 美音男4
4 - 美音男5
5 - 美音女1
6 - 美音女2

8 - 美音女4（默认）
